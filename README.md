# kmeans-credit-card-clustering
we have dataset of +10k client with some features like (age, marital_status, ...)
and we want to group customers into different "personas" based on their
spending behavior and credit history.


### Ressources:
Unsupervised Learning: https://www.geeksforgeeks.org/machine-learning/unsupervised-learning/
K-Means Clustering Explination: https://www.youtube.com/watch?v=6JQK03IGIvU


# DataSet
varaibles explination:
- dependent_count: how many childre he have
- months_on_book: number of months the customer has been with the company.
- total_relationship_count: how many products client uses.
- months_inactive_12_mon: number of month customer does not use credit card for past 12 months.
- credit_limit: with credit card how much many able to spend.
- total_trans_amount: total money the customer spent.
- total_trans_count: how many transaction did
